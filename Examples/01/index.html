Exprimiendo JavaScript

  1. Abrir la consola usando

    CTRL + F12

  2. En la consola, escriba y ejecute

    document

    para acceder al documento

    El resultado obtenido es el objeto document

    #document

  3. Escriba y ejecute

    document.getElementById('titulo')

    El resultado obtenido es:

    <‍h1 id="titulo" class="title">Exprimiendo JavaScript<‍/h1>
    Aquí se usa el DOM para buscar un elemento por nombre identificador

  4. Ejecutar

    document.getElementsByClassName('title')[0].style.color="yellow"

    Verá que el título de la página, ha cambiado de color. El resultado obtenido es:

    Exprimiendo JavaScript

    Aquí se usa el DOM para buscar un elemento por nombre de clase y cambiar su estilo

  5. Ingresar al archivo script.js

    En el archivo se encuentra la función toggle, invocada cuando se hace clic en uno de los botones.

    Cuando se ejecuta la función, se busca por nombre de etiqueta <‍li> y se obtiene el i-esimo elemento de la lista que corresponda al índice index. Aquí se usa el DOM para buscar un elemento por nombre de etiqueta

    Por último, obtiene el primer elemento que contenga la clase result y setea el atributo hidden del objeto. Aquí se usa el DOM para buscar un elemento por clase y para cambiar el valor de un atributo
